# Product Register

This project is a sophisticated AI agent "builder" documentation site. 
Tone: Technical, highly authoritative, developer-focused, AI-native.
Target Audience: Senior engineers, AI researchers, platform developers.
Core principles: Clarity over cleverness, immediate path to value, dark-mode first, stark contrast.
Anti-patterns: Playful illustrations, excessive emojis, bright pastel colors, "friendly" SaaS copy.
